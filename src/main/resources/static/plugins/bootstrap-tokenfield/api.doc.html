tokenfield参数配置

参数 默认值 类型 备注
limit 0 int 可输入标签个数，0==无限制
createTokensOnBlur false boolean 焦点失去后是否创建token
delimiter ',' string, array 标签分隔符

tokenfield方法说明

方法名称 参数 参数类型 描述 备注
tokenfield options：用户自定义配置
disable,
enable,
destroy,
readonly,
writeable 		String,
Json对象 		开启tokenfield 
  $('#myField').tokenfield({
    limit: 5,
    delimiter: [',','，'],
    createTokensOnBlur: true
  });
  $('#myField').tokenfield('disable');
  $('#myField').tokenfield('enable');
  $('#myField').tokenfield('readonly');
  $('#myField').tokenfield('writeable');
  $('#myField').tokenfield('destroy');
setTokens tokens：input.value string,
array,
jsonArray		 更新input值 
  $('#myField').tokenfield('setTokens', 'blue,red,white');
  $('#myField').tokenfield('setTokens', ['blue','red','white']);
  $('#myField').tokenfield('setTokens', [{
                value: "blue",
                label: "Blau"
              }, {
                value: "red",
                label: "Rot"
              }]
            );
createToken token:input-value String,
Json		 创建新的token 
  $('#myField').tokenfield('createToken', 'purple');
  $('#myField').tokenfield('createToken', {
                value: 'violet',
                label: 'Violet'
              });
getTokens 获取token值 
  $('#myField').tokenfield('getTokens');
  $('#myField').tokenfield('getTokens', active);
getTokensList delimiter String 将获取到的token值以delimiter分隔 
  $('#myField').tokenfield('getTokensList');
  $('#myField').tokenfield('getTokensList', '; ');

tokenfield事件

事件名称 描述 备注
tokenfield:initialize Tokenfield初始化发生时
tokenfield:createtoken Tokenfield创建之前 e.attrs
阻止创建：event.preventDefault() 或 return false
tokenfield:createdtoken Tokenfield创建成功后 e.attrs
创建对象：e.relatedTarget
tokenfield:edittoken Tokenfield编辑发生前 e.attrs
阻止创建：event.preventDefault() 或 return false
tokenfield:editedtoken Tokenfield编辑发生后 e.attrs
创建对象：e.relatedTarget
tokenfield:removetoken Tokenfield删除之前 e.attrs
阻止删除：event.preventDefault() 或 return false
tokenfield:removedtoken Tokenfield删除成功后 e.attrs
对象：e.relatedTarget
使用举例 
   $('#tokenfield').on('tokenfield:createtoken', function (e) {
      var data = e.attrs.value.split('|')
      e.attrs.value = data[1] || data[0]
      e.attrs.label = data[1] ? data[0] + ' (' + data[1] + ')' : data[0]
    })

    .on('tokenfield:createdtoken', function (e) {
      // Über-simplistic e-mail validation
      var re = /\S+@\S+\.\S+/
      var valid = re.test(e.attrs.value)
      if (!valid) {
        $(e.relatedTarget).addClass('invalid')
      }
    })

    .on('tokenfield:edittoken', function (e) {
      if (e.attrs.label !== e.attrs.value) {
        var label = e.attrs.label.split(' (')
        e.attrs.value = label[0] + '|' + e.attrs.value
      }
    })

    .on('tokenfield:removedtoken', function (e) {
      alert('Token removed! Token value was: ' + e.attrs.value)
    })
